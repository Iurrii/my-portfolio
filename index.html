*Сайт находится в разработке

В данный момент накопилось несколько проектов "в стол", и чтобы не прикреплять к резюме пачку ссылок, сделал этот сайт-каталог со своими проектами

Погодный виджет

Репозиторий

HTML, CSS, JS, API, HTML-in-JS.
Отработка навыков работы с API+Json данными - асинхронный запрос, получение ответа, вставка полученных данных на страницу. Анимация fontawesome. Работа с Object.Date и часовым поясом активного устройства. Для почти каждого ЧП город по умолчанию из соответствующего ЧП.

Калькулятор на чистом JS

HTML, CSS, JS.
Простой калькулятор на чистом JS. Работает на 3х переменных, поддерживает каскадные операции типа, "а+б-с". Использование сторонних иконок и шрифтов (fontawesome, a_LCDNova), grid вёрстка. Уникальный дизайн.

Сайт обмен валют (React)

HTML, CSS, JS, React, API.
Создание React SPA. Работа с компонентами, роутинг. Отработка навыков работы с API+Json данными - асинхронный запрос, получение ответа, вставка полученных данных на страницу. Калькуляция.

Крестики-нолики (React)

HTML, CSS, JS, React, localStorage.
"Крестики-нолики" на двух игроков. Использование localStorage для хранения счёта побед. Опция выбора стартового символа, обнуление счётчика побед.

Интерфейс чата

HTML, CSS.
Flex-вёрстка адаптивного дизайна для интерфейса чата. Адаптивный дизайн, "резиновые" блоки, с прокруткой содержимого.

Разное на "CodePen"

Заготовки и фрагменты кода, не "тянущие" на самостоятельный проект, но представляющие определённый интерес.
Скрипты, шаблоны, input'ы, уже выполненные тестовые задания.
