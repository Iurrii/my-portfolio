Погодный виджет

Репозиторий

HTML, CSS, JS, API.
Отработка навыков работы с API+Json данными - асинхронный запрос, получение ответа, вставка полученных данных на страницу. Генерация HTML разметки из массива. Анимация fontawesome. Работа с Object.Date и часовым поясом активного устройства. Для почти каждого ЧП город по умолчанию из соответствующего ЧП.

Калькулятор на чистом JS

HTML, CSS, JS.
Простой калькулятор на чистом JS. Работает на 3х переменных, поддерживает каскадные операции типа, "а+б-с". Иконки fontawesome, grid вёрстка.

Сайт обмен валют (React)

HTML, CSS, JS, React, API.
Создание React SPA. Работа с компонентами, роутинг. Отработка навыков работы с API+Json данными - асинхронный запрос, получение ответа, вставка полученных данных на страницу. Калькуляция.

Крестики-нолики (React)

HTML, CSS, JS, React, localStorage.
"Крестики-нолики" на двух игроков. Использование localStorage как хранилище для счёта побед. Опция выбора стартового символа.

Интерфейс чата

HTML, CSS.
Flex-вёрстка адаптивного дизайна для интерфейса чата. Адаптивный дизайн, "резиновые" блоки, с прокруткой содержимого.

Разное на "CodePen"
